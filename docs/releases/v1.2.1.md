Status: planned prerelease from `main` after the 1.2.1 release-prep PR merges.

Base release: `v1.2.0`

Release target: the merged `main` commit that includes this release-prep note.
Do not tag an earlier stabilization branch commit if the release is meant to

Asset: `qortium-preview.zip`

## Summary

This prerelease packages the Previewnet stabilization fixes found during live
seed-node, I2P, and archive fast-sync testing after `v1.2.0`.

## Highlights

- I2P fallback recovery: chain and data SAM fallback sessions now detect an
  unexpected SAM control disconnect and rebuild instead of staying down until
  another operation happens to touch I2P.
- Archive fast-sync replay fix: non-AT transactions from archived blocks are
  saved before replay links them to their block, preserving the normal sync
  invariant and preventing the `BlockTransactions` foreign-key failure seen on
  fresh-node replay.
- Replay status progress: local node status and tray reporting can show replay
  height moving during archive replay without advertising that uncommitted
  height to peers before the atomic checkpoint-gated replay commits.
- Qortal 6.1.8 validation follow-ups: grouped online-account and trade-presence
  peer messages validate every group count, `NamesMessage` uses a minimum
  entry-size precheck, and AT MESSAGE deserialization accepts the existing
  256-byte validator limit while rejecting invalid declared lengths.
- Jackson advisory tracking: OSV scanner configuration records the temporary
  ignore for CVE-2026-54515 while Jackson's patched 2.x artifacts are not yet
  available from Maven Central.

## Notes

- This is a public preview prerelease. Preview accounts, data, and balances are
  not permanent.
- The I2P fallback transport remains optional and configured through the
  ordered `allowedTransports` setting; nodes default to direct TCP
  connectivity. Set `allowedTransports` to `["I2P"]` for an I2P-only node.
- The Jackson OSV ignore should be removed as soon as patched Jackson 2.x
  artifacts are available and the dependency bump builds successfully.
- Post-release update (2026-07-14): Jackson `2.22.1` has now been adopted and
  carries the fix for CVE-2026-54515. The ignore remains temporarily only
  because scanner metadata still flags the fixed version, not because patched
  artifacts are unavailable.
- Upload `target/qortium-preview.zip` to the GitHub draft before publishing.
